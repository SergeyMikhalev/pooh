###  Проект job4j_accidents - Учет дорожных правонарушений

### Краткое описание
Приложение предоставляет средства для ведения учета
дорожных правонарушений силами граждан.

### Стэк технологий
- Core: Java 17, Maven 3.6.3.

### Окружение
Для запуска проекта понадобятся
- Java 17
- Maven 3.6
