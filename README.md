###  Проект job4j_pooh - Сервер сообщений

### Краткое описание
Приложение предоставляет возможность обмена сообщениями.
Работает в двух режимах queue и topic

### Стэк технологий
- Core: Java 19, Maven 3.6.3.

### Окружение
Для запуска проекта понадобятся
- Java 18
- Maven 3.6
